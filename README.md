# 🚀 WebArtUpStudio

An all-in-one web utility platform that brings together multiple productivity tools into a single modern dashboard. Built with performance, simplicity, and scalability in mind, WebArtUpStudio helps users accomplish everyday digital tasks without switching between multiple websites.

🌐 **Live Demo:** https://artup.pages.dev/

---

## ✨ Features

### 🖼️ Photo Converter
- Convert images between multiple formats.
- Fast processing optimized for web usage.
- Lightweight and responsive interface.

### 🔗 Link Scanner
- Analyze URLs before visiting them.
- Detect potentially suspicious or malicious websites.
- Improve browsing safety.

### 🤖 AI Humanizer
- Rewrite AI-generated text into more natural, human-like writing.
- Clean and readable output.
- Designed for content creators and professionals.

### 📱 Responsive Design
- Mobile-first responsive layout.
- Optimized for desktop, tablet, and mobile devices.
- Modern UI built with Tailwind CSS.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/pdroid908/webartupstudio.git
```

Move into the project directory

```bash
cd webartupstudio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Run production mode

```bash
npm start
```
